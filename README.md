# Memory-card-game
An interactive memory card game that challenges players to match pairs by flipping cards, developed using HTML, CSS, and JavaScript.

## Features

* Card flip animation
* Matching card logic
* Timer to track gameplay
* Multiple difficulty levels
* Responsive and simple interface

## Tech Stack

* HTML
* CSS
* JavaScript

## Gameplay

1. Select a difficulty level.
2. Flip two cards at a time.
3. If the cards match, they remain open.
4. If they do not match, they flip back.
5. Match all pairs before the timer ends.
